## Install

You can download or `git clone` this repo

```sh
$ git clone https://github.com/kpman/starter new-project
$ cd new-project
$ rm -rf .git
$ yarn # or npm install
```

Just make sure to edit `package.json`, `README.md` and `LICENSE` files accordingly with your module's info.
